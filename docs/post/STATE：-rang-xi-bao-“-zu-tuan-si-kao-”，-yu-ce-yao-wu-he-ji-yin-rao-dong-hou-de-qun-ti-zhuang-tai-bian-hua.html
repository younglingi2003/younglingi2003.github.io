STATE：让细胞“组团思考”，预测药物和基因扰动后的群体状态变化

一句话概括

STATE 想做的是：不用真的把每种药、每个基因都拿去做实验，先让 AI 预测“一群细胞受到刺激后，会变成什么状态”。 这篇论文于 2025 年发布在 bioRxiv，目前属于预印本。(生物学预印本)

它要解决什么问题？

假设研究人员想知道：

给癌细胞使用某种药物后，细胞会发生什么变化？

传统方法需要真正培养细胞、加入药物，再测量细胞中的 RNA。问题是，药物、基因、细胞类型和剂量的组合太多，不可能全部做一遍。

STATE 希望先在电脑里进行“虚拟实验”：

  • 输入：处理前的细胞，以及要施加的药物、细胞因子或基因改动；
  • 输出：处理后，这些细胞的基因表达可能发生什么变化。(Arc Institute)

这里的“基因表达”可以理解成细胞内部的一张工作状态表：哪些基因正在积极工作，哪些基因比较安静。

STATE 最特别的地方

以前很多模型会把细胞一个一个单独处理，但现实中，即使是同一种细胞，面对同一种药物，反应也不完全一样：

  • 有些细胞反应强；
  • 有些细胞反应弱；
  • 有些可能处于分裂阶段；
  • 有些甚至几乎不响应。

STATE 不只观察一个细胞，而是同时观察一群细胞。它让模型比较这些细胞之间的差异，再预测整个细胞群体经过处理后的分布。

可以把它理解为：

以前的模型是在预测“一个学生考试后能得多少分”；
STATE 则是在观察整个班级，预测考试后全班成绩会怎样分布。

这种基于细胞集合的注意力机制，可以帮助模型保留细胞群体中的异质性，而不是把所有细胞简单平均掉。(Arc Institute)

它由两个部分组成

1. State Embedding：给细胞建立“坐标系”

它读取一个细胞的 RNA 数据，把复杂的基因表达压缩成一组数字坐标。

状态相似的细胞会靠得比较近，例如同类免疫细胞可能聚在一起；状态不同的细胞则相距较远。这个模块主要负责回答：

“这个细胞现在处于什么状态？”

2. State Transition：预测细胞如何移动

第二个模块接收：

  • 处理前的一群细胞；
  • 某种药物或基因扰动；
  • 相关细胞类型和实验条件。

然后预测这些细胞会在前面的“状态坐标系”中如何移动，也就是：

“受到处理后，这群细胞可能变成什么样？”(Arc Institute)

一个具体例子

假设模型看到一群白血病细胞，研究人员输入一种新药。

STATE 可能预测：

  • 某些与细胞分裂有关的基因会被关闭；
  • 某些压力反应基因会被打开；
  • 大部分细胞停止增殖；
  • 少部分细胞反应较弱，可能形成耐药亚群。

它输出的并不是简单的“药物有效”或“药物无效”，而是一批预测出来的处理后细胞，以及它们的基因表达状态。

论文做出了什么结果？

作者使用了约 1.7 亿个观察性细胞训练细胞表示模块，并使用超过 1 亿个经过药物、基因或细胞因子扰动的细胞训练状态转移模块，数据覆盖约 70 个细胞系。(Arc Institute)

在论文的测试中，STATE 被要求预测模型没有完整见过的“细胞类型—扰动”组合。作者报告称，相比已有方法和简单基线，它更擅长：

  • 区分不同药物或基因造成的影响；
  • 找出哪些基因会明显升高或降低；
  • 保留同一群细胞内部不同的反应；
  • 将学到的扰动规律迁移到新的细胞环境。(Arc Institute)

这项工作有什么意义？

它最现实的用途不是直接替代实验，而是帮助研究人员缩小实验范围。

例如原本有一万种药物或基因改动需要测试，STATE 可以先在电脑中预测，筛选出几十个最有希望的候选，再让研究人员真正做实验。这可能用于：

  • 药物筛选；
  • 寻找潜在治疗靶点；
  • 预测癌细胞的耐药反应；
  • 研究某个基因被关闭后的影响；
  • 寻找让病变细胞恢复健康状态的方法。(Arc Institute)

需要注意什么？

STATE 还不是一个能够完全模拟真实细胞的“数字生命”。

它目前主要预测的是 RNA表达变化，不能完整预测蛋白质、细胞形态、代谢、组织环境以及患者真实疗效。模型也会受到训练数据覆盖范围、实验批次和测量噪声影响。

Arc Institute 后续开展的虚拟细胞挑战也显示，现有扰动预测模型尚不能在所有评价指标上稳定超过简单基线，尤其是精确还原每个基因的表达数值仍然很困难。因此，STATE 更适合充当实验假设生成器和候选筛选器，而不是实验结果的替代品。(Arc Institute)

最通俗的理解

STATE 就像一个细胞版的天气预报模型：它读取一群细胞现在的状态，再预测加入药物或修改基因后，这群细胞的“内部天气”会怎样变化。

可以给这篇论文概括为：

STATE：让一群细胞彼此“交流”，预测药物和基因扰动后的状态变化。
